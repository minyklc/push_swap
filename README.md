**--- Description ---**

push_swap is a program in c that sorts data with 2 stacks and a set of instructions to manipulate both stacks, in an optimized way.
the output is the set of operations needed to organize the input.

**--- Instructions ---**

`make`

`./push_swap <set of data>` (needed instructions) (example: "0 1 2 5 4")

`./push_swap | wc -l` (number of needed instructions)

`make fclean`
